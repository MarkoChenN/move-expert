# 第1节：安装环境

本节主要完成move开发环境搭建

## 下载工程

```sh
git clone https://github.com/move-language/move.git
```

## 配置环境

```sh
cd move
./scripts/dev_setup.sh -ypt
```

## 安装move-cli

```sh
cargo install --path language/tools/move-cli
```

## 测试

```sh
move --help
```
